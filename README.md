# mvsep_app
MVSep application

```
# Получить типы разделения
main.exe get_types
Выполнение данной команды выведет все доступные типы разделений.

# Создать раделение с заданными параметрами
main.exe create_separation path/to/file.mp3 your_api_token separation_type add_opt1 add_opt2
API_TOKEN можно найти в личном кабинете на сайте [mvsep](https://mvsep.com/en)
Параметры separation_type, add_opt1 и add_opt2 можно получить сделав первый запрос,
если в типе нет одного из дополнительных параметров (add_opt1 или add_opt2), укажите на его месте 0. 

# Получить результат (файлы) разделения на рабочий стол
main.exe get_result hash
Выполнение данной команды, загрузит итоговые файлы на рабочий стол,
если они уже готовы, в противном случае вы получите ошибку говорящую о том,
что файлы ещё не готовы.
```
